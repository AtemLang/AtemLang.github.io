Inspirations List

This list contains the language/library features that are inspired by other languages/libraries.

C++​

  • Basic design principles
  • Generics
  • Compile-time metaprogramming

Cpp2/Cppfront​

  • Uniform declaration syntax name : declarator = body
  • Type expression

Circle/New Circle​

  • Traits
  • Concept/Implement

Carbon​

  • Generics
  • Operator precedence

Rust​

  • Concept-based operator overloading
  • Package Management System

Swift​

  • Arrays
  • Exception Handling
  • Function

D​

  • Arrays
  • Optional

Zig​

  • Types
  • Optional
  • Variables: No unused local variable, no identifier shadowing

Scala​

  • Contextual abstraction
  • Intersection types

Kotlin​

Haskell​

Julia​
